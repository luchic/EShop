# Project Intro

We are building a digital marketplace where users can spend their internal currency to buy digital products or services.

This keeps the project inside the rule that only digital objects or services are allowed.

## Currency
Users have a platform balance.
This balance can be connected to intra-related points or evaluation rewards, depending on the final project rules.

Users can:
- view their current balance
- spend balance on digital items
- review their purchase history

## Main Features
- Product catalog with name, description, price, and availability
- User account page with balance and purchase history
- Product page for item details
- Shopping cart / checkout flow
- Transaction log for purchases and refunds
- Admin panel for managing products, availability, and refunds

## Anti-Abuse
- input validation on all forms
- rate limiting for sensitive actions
- protection against invalid or duplicate purchases
- permission checks for admin actions

## Tech Stack
- Frontend: React
- Backend: Go
- Separate frontend and backend services
